
# World Cup Matches Analysis

This project performs various analyses on World Cup match data, including visualizations of team goals, distributions, and comparisons of teams over different years.

## Tasks

1. **Density Curve**: Distribution of Home Team Goals for each year.
2. **Violin Plot**: Distribution of Home Team Goals by year with selective labels.
3. **Joyplot**: Distribution of Home and Away Team Goals across years.
4. **Grouped Data**: Bar plot for total Home Team Goals for the top 5 countries.
5. **QQ Plot**: Comparison of Home Team Goals and Away Team Goals distributions.

## Libraries Used
- pandas
- matplotlib
- seaborn
- joypy
- seaborn_qqplot

## Installation Instructions
1. Install the required libraries using pip:
   ```bash
   pip install pandas matplotlib seaborn joypy seaborn-qqplot
   ```
2. Run the script `WorldCupMatches_Assignment.py` to perform the analysis.
